# 项目独立开发说明 / Independent Development Guide

## 中文说明

本项目最初是从其他仓库fork而来，但现在已经决定独立开发，不再与原始仓库保持同步。

### 🎯 关键要点

- **不再同步原始仓库** - 本仓库将不会合并原仓库的更新
- **独立发展方向** - 当前开发以 `docs/epubpro-v1-plan.md` 为主计划，`docs/archive/plan-epubToolsV2.prompt.md` 作为历史参考
- **Git 配置正确** - 远程仓库已指向您自己的仓库
- **双语文档** - 提供中英文文档

### 未来开发计划

项目未来的发展方向请优先参考 `docs/epubpro-v1-plan.md`，历史演进背景可参考 `docs/archive/plan-epubToolsV2.prompt.md`，主要包括：

- 使用 TypeScript 重写核心功能
- 采用 Tauri 构建桌面应用
- 仅保留 Python 的字体混淆功能
- 提供 CLI 和 GUI 双界面

## English Guide

This project was originally forked from another repository but has now decided to develop independently without synchronizing with the original repository.

### 🎯 Key Points

- **No longer synchronize with original source** - This repository will not merge updates from the original repository
- **Independent development direction** - Current development follows `docs/epubpro-v1-plan.md`, with `docs/archive/plan-epubToolsV2.prompt.md` kept as historical reference
- **Git configuration is correct** - Remote repository points to your own repository
- **Bilingual documentation** - Documentation provided in both Chinese and English

### Future Development Plan

For future development direction, please refer to `docs/epubpro-v1-plan.md` first. Historical context remains in `docs/archive/plan-epubToolsV2.prompt.md`, which includes:

- Rewriting core functionality in TypeScript
- Building desktop applications with Tauri
- Keeping only Python font obfuscation functionality
- Providing both CLI and GUI interfaces
